Car Database Website 🚗

This is a web application built using Flask that allows users to view and manage a car inventory.

Features
View car listings
Add new cars
Store data in a database
Simple UI using HTML/CSS
How to Run
Clone the repository:
git clone https://github.com/Calvin2334/CarDataBaseWebsite.git
Go into the folder:
cd CarDataBaseWebsite
Install dependencies:
pip install -r requirements.txt
Run the app:
python app.py
Open your browser and go to:
http://127.0.0.1:5000/
